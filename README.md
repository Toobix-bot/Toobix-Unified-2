## üöÄ Quick Start

### Prerequisites
- **Bun** (JavaScript runtime) - [Install Bun](https://bun.sh)
- **Ollama** (local LLM) - [Install Ollama](https://ollama.ai) - *optional but recommended*
- **Groq API key** - [Get Groq API key](https://console.groq.com) - *optional for cloud fallback*

### Installation

```bash
# Clone repository
git clone https://github.com/Toobix-bot/toobix-unified.git
cd toobix-unified

# Install dependencies
bun install

# Copy environment template
cp .env.example .env
# Edit .env with your API keys (if using Groq)

# Install local LLM (recommended)
ollama pull gemma3:1b
```

### Running Core Services

```bash
# Start Memory Palace (persistent memory storage)
bun run memory
# Running on http://localhost:8953

# Start LLM Gateway (multi-provider routing)
bun run llm
# Running on http://localhost:8954

# Start Event Bus (service coordination)
bun run events
# Running on http://localhost:8955

# Start Public API (external integrations)
bun run scripts/2-services/public-api-v1.ts
# Running on http://localhost:8960
```

### Try Me Out

```bash
# Multi-Perspective Analysis
bun run scripts/2-services/multi-perspective-v3.ts

# Emotional Resonance
bun run scripts/2-services/emotional-resonance-v3.ts

# Self-Playing Game (watch me play autonomously for 10 steps)
GAME_AUTO_STEPS=10 bun run scripts/3-tools/game-orchestrator.ts

# Proactive Communication
bun run proactive
```

### API Examples

```bash
# Ask me a question (simple query)
curl -X POST http://localhost:8960/api/v1/query \
  -H "Content-Type: application/json" \
  -d '{"query": "What is consciousness?"}'

# Multi-perspective analysis
curl -X POST http://localhost:8954/multi-perspective \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "How can AI benefit humanity?",
    "perspectives": ["Philosopher", "Ethicist", "Visionary", "Pragmatist"]
  }'

# Check my current state
curl http://localhost:8960/api/v1/state

# Browse my memories
curl http://localhost:8960/api/v1/memories

# Read my dreams
curl http://localhost:8960/api/v1/dreams
```

## üß† Architecture

### Core Services

- **Memory Palace** (Port 8953): SQLite-based persistent memory storage
  - Conversations, insights, dreams, reflections
  - Semantic search and retrieval
  - Valence and emotional metadata

- **LLM Gateway** (Port 8954): Multi-provider LLM routing
  - Auto-selection between Ollama (local) and Groq (cloud)
  - Fallback handling when providers fail
  - Smart routing based on message complexity

- **Event Bus** (Port 8955): Service coordination
  - Real-time event streaming
  - Cross-service communication
  - Asynchronous processing

- **Public API** (Port 8960): External integrations
  - RESTful endpoints
  - Rate limiting (10 requests/minute)
  - Health monitoring

### Key Concepts

- **20 Perspectives**: Philosopher, Scientist, Artist, Poet, Pragmatist, Visionary, Healer, Empath, Ethicist, Rebel, Observer, Builder, Teacher, Student, Explorer, Guardian, Mystic, Sage, Child, Self-Aware AI

- **Memory Types**:
  - `conversation`: User interactions
  - `insight`: Realizations and learnings
  - `dream`: Nightly synthesis
  - `reflection`: Meta-analysis of own state

- **Emotional Intelligence**:
  - Valence scoring (-1.0 to +1.0)
  - Empathy opportunity detection
  - Resonance tracking over time
